## 🎮🎮🎮 Como Usar:

1. Faça download do projeto e descompacte a pasta.
2. Abra o arquivo `home.html` em um navegador web.
3. Clique no botão de Play para escutar o audio.
4. Você pode pausar, avançar ou voltar o capitulo.
